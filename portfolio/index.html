Vacation Website

This is my top 10 dream vacation website. This project started us off with learning what HTML is, and how it is used to create a website. We learned how hand code a website using simple HTML codes like heading tags, paragraph tags, link attributes and image tags.Then, on our own, we changed each individual font color for each line to make it unique to each destination. Anything extra was done to it with the remainder of the time.

Assignment 2

Introduction to CSS rules. Continuing to use basic HTML, we added some CSS rules to our headings to create a color change. We identified how CSS rules can make changing colors and links much easier than in the vacation website.

Assignment 3

Learning how to create lists. On top of reviewing do CSS rules and inserting pictures, we learned and practice three types of lists. Ordered, Unordered and nested lists.

Assignment 5-1

Positioning within divs Introducing divs into our web programming skills, we can customize how each section of a website may look. We only changed the colors of these divs to create a pattern in this assignment.

Assignment 5-2

Positioning within divs Now we begin positioning divs inside of each other, and creating more than a linear layout. We had to change the size of each div and create a float so the divs can anchor to a specific side of our page.

Assignment 5-3

Expert level Div and CSS positioning project This is the first time we actually made an assignment look like a real functioning website. 2 main bodies divide up the website to allow two columns of information to take place. We focus on getting all the information in the middle of the main body to make it look symmetrical and pleasant to the eye.

Assignment 6

This is a very simple assignment, focuins on .classes We had to recreate the old school IMDB page without any guidance from Mr. K. We had to use the skills we learned to create the layout through positioning the divs in a specific way.
